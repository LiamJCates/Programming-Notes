# emplace_back

**Description**: Constructs and places an element at the end of the vector

**Example**:
```cpp
    //creating a vector of 3 elements
    std::vector<triagle> vec{10, 20, 30};

    //function to construct and insert a element at the end of the vector
    vec.emplace_back(40);
    vec.emplace_back(50);
```
