# swap

**Description** : Swaps the contents of two vectors

**Example**:
```cpp
    //Intializing the vectors  
    std::vector<int> vector1 = {1,2,3};
    std::vector<int> vector2 = {4,5,6,6};

    //Function to swap the vectors
    vector1.swap(vector2);

```
